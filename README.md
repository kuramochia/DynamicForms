# DynamicForms
Forks from https://github.com/MobileEssentials/DynamicForms and Re-create .NET Standard 2.0 version.

## Install
### Create .nupkg
Please use MSBuild version 15.1+

    > msbuild /t:pack /p:Configuration=Release Xamarin.Forms.Dynamic.csproj
